# Support

This is the organization-wide default support guide for FAST projects. An
individual repository may provide its own `SUPPORT.md` that takes precedence.

## Getting help

- **Read the docs first** — check the repository's `README.md` and any linked
  documentation or reference material.
- **Search existing issues** — your question may already be answered in the
  repository's [issues](../../issues).
- **Ask a question** — if you can't find an answer, open an issue in the relevant
  repository. Include enough detail (what you're trying to do, what you tried,
  your environment or hardware revision) for others to help you.

## What is not a support channel

Please don't use the issue tracker for security or safety reports — see
[SECURITY.md](SECURITY.md) for how to report those privately.
